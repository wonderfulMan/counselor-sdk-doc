# 视图类接口（view）

# closeNewView

  • 说明：关闭当前 view
  • 参数 payload 对象如下：
    • refresh {boolean} 是否刷新上级页面 默认：false
    • toDeep {number} 返回到上几级页面 默认：1
  • tips：此 Api 参数只支持今日校园，其中toDeep参数：返回上级传 1，上上级传 2，上上上级传 3，最外层传 0(即退出所有 webView)，退到所有 webView 的根 webView 传-1(即只保留第一个 webView)，超出页面深度相当于 0
  • 返回值：
    • 成功：{sdkApiReturnSuccessResponse}
    • 失败：{sdkApiReturnErrorResponse|sdkEnvErrorResponse}
  • code 返回范围：0/-1
import { closeNewView } from "counselor-sdk";

const { code } = await closeNewView();

# openNewView

  • 说明：打开一个 view
  • 参数 payload 对象如下：
    • url {string} 地址
  • tips：此 Api 参数只支持今日校园
  • 返回值：
    • 成功：{sdkApiReturnSuccessResponse}
    • 失败：{sdkApiReturnErrorResponse|sdkEnvErrorResponse}
  • code 返回范围：0/-1
import { openNewView } from "counselor-sdk";

const { code } = await openNewView();

# setNavHeader

  • 说明：设置顶部 bar
  • 参数 payload 对象如下：
    • title {string} 窗口标题
    • left {object} 左上角按钮
      • title {string} 左侧按钮标题
      • callFunction {function} 左侧按钮回调
    • right {object} 右侧按钮组配置
      • title {string} 右侧按钮组标题
  • tips：此 Api 参数只支持今日校园
  • 返回值：
    • 成功：{sdkApiReturnSuccessResponse}
    • 失败：{sdkApiReturnErrorResponse|sdkEnvErrorResponse}
  • code 返回范围：0/-1/-7
import { setNavHeader } from "counselor-sdk";

const { code } = await setNavHeader({
  title: "hi guys",
  left: {
    title: "xxxx",
    callFunction: () => {
      console.log("我被重写了");
    },
  },
  right: {
    title: "xxxx",
  },
});

# setViewTitle

  • 说明：设置窗口标题
  • 参数 payload 对象如下：
    • title {string} 窗口标题
  • tips：此 Api 微信中采用 h5 实现
  • 返回值：
    • 成功：{sdkApiReturnSuccessResponse}
    • 失败：{sdkApiReturnErrorResponse|sdkEnvErrorResponse}
  • code 返回范围：0/-1
import { setViewTitle } from "counselor-sdk";

const { code } = await setViewTitle({ title: "hi guys" });

# webviewOnResume

  • 说明：创建当前窗口 Resume 监听
  • 参数 payload 对象如下：

  • tips：此 Api 微信无实现
  • 返回值：
    • 成功：{sdkApiReturnSuccessResponse}
    • 失败：{sdkApiReturnErrorResponse|sdkEnvErrorResponse}
  • code 返回范围：0/-1/-7
import { webviewOnResume } from "counselor-sdk";

const { code } = await webviewOnResume({ callback: () => {} });

# goToAppHome

  • 说明：跳转至平台首页
  • 参数 payload 对象如下：
    • from {string} 从哪来 默认：'CAT'
    • tabIndex {number} 选中tab索引 从0开始 默认：0
  • tips：此 Api 微信无实现，今日校园直接返回首页，该Api有默认值建议不传。
  • 返回值：
    • 成功：{sdkApiReturnSuccessResponse}
    • 失败：{sdkApiReturnErrorResponse|sdkEnvErrorResponse}
  • code 返回范围：0/-1/-7
import { goToAppHome } from "counselor-sdk";

const { code } = await goToAppHome();
