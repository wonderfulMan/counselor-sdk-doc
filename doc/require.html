# 使用须知

  1. 以下 Api 接口描述中，微信统称微信公众号、企业微信，参数无标注必传均为选传（选传含有默认值会在具体Api进行标注）
  2. code 返回范围均如下
OK: 0, // 所有api成功返回结果
ERROR: -1, // 当内部发生异常时
CANCEL: -2, // 当照相/相册api会触发
INIT_ERROR: -3, // sdk初始化失败
GET_LOCATION_ERROR: -4, // 获取经纬度失败
LOACTION_PERMISSIONS: -5, // 定位权限未开启或者没有获取到
NOT_FOUND_BMAP: -6, // 百度地图加载失败或者未加载
NOT_WX_ALL_IMPLETEMENT: -7, // 微信、企业微信无实现时返回
NOT_WX_WORK_IMPLETEMENT: -8, // 企业微信无实现返回

需要使用该常量请引入import { CODE } from 'counselor-sdk'

  1. 返回值解析
    • sdkApiReturnSuccessResponse 类返回：
{
  code: 0,
  data: payload || null,// 无需返回均为null，有则根据Api返回值代码示例
  message: 'success' // 默认success
}
  • sdkApiReturnErrorResponse 类返回：
{
  code: -1, // 无特殊场景均为-1
  data: null,// 无需返回均为null，有则根据Api返回值代码示例
  message: `调用环境{platform}：{apiName}返回结果异常{err}`,
  // platform 微信或今日校园
  // apiName 接口名称
  // err 可能是微信、今日校园抛出的错误描述，或者无法判断时给出的提示
}
  • sdkEnvErrorResponse 类返回：
{
  code: -1, // 无特殊场景均为-1
  message: `调用环境{platform}: 当前调用api名称{apiName}，请检查sdk是否初始化成功`,
  // 通常为 sdk初始化失败或者，wx/mamp在当前宿主环境上没有找到，该类型会在自动生产模版时，自动注册，无需
  // 手动添加
}

  1. 部分接口采用了 h5 实现，对于定位类地址解析使用了百度地图相关 Api，使用时请确保你的应用成功加载了百度地图 jssdk（建议百度地图版本为 2.0）

  2. 该 Api 版本均为辅导猫内部开发使用，对于外部使用出现的问题，优先级会降低处理，请见谅，接受优化意见！

  3. 所有 Api 打包均根据具体项目走，请在项目方使用时注意是否忽略了 node_modules 文件夹，如果忽略了请设置 counselor-sdk 包名
