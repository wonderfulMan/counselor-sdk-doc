# 定位类接口（location）

# getCurrentAddress

说明：根据经纬度返回地址名称（内部调用百度地图进行转换，业务中必须加载百度地图进行使用）

  • 参数 payload 对象如下：
    • longitude {number} 必传
    • latitude {number} 必传
    • timeout {number}
    • whenTimeoutAddress {string}
  • tips：
    • 逆地址解析超时默认值为 5s，你可以通过指定 timeout 来设置你的超时时间，当超时时
      会返回 whenTimeoutAddress 设置当默认值，如果该值不存在默认为签到地点
  • 返回值：
    • 成功：{sdkEnvErrorResponse}
    • 失败：{sdkApiReturnSuccessResponse｜sdkApiReturnErrorResponse}
  • code 返回范围：0/-1/-4
import { getCurrentAddress } from "counselor-sdk";

const { code, data } = await getCurrentAddress({
  longitude: 111,
  latitude: 2222,
  timeout: 5,
  whenTimeoutAddress: "暂无地址",
});

const address = data; // 成功时为获取的地址

# getCurrentPosition

  • 参数：无
  • 返回值：
    • 成功：{sdkApiReturnSuccessResponse}
    • 失败：{sdkApiReturnErrorResponse|sdkEnvErrorResponse}
  • code 返回范围：0/-1/-6
import { getCurrentPosition } from "counselor-sdk";

const { code, data } = await getCurrentPosition();

const { longitude, latitude } = data; // 成功时为经纬度对象
