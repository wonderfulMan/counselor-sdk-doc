# counselor-sdk 启动函数

# bootstrap

  • 说明：启动 counselor-sdk
  • 参数：
    • request {function} 微信中获取参数配置请求函数
    • options {object}选传
      • wxDebugger {boolean} 是否在开发环境中开启微信 debbuger 模式
        • 默认：开发阶段默认值为true
      • initCallback {funtion} 初始化完成后立即调用的回掉函数
    • headers {object} 请求头
  • tips：此函数应该在调用任意 Api 前进行调用，将完成一系列初始化、注入脚本、配置平台 sdk 信息等
  • 返回值：
    • 成功：{sdkApiReturnSuccessResponse}
    • 失败：{sdkApiReturnErrorResponse|sdkEnvErrorResponse}
  • code 返回范围：0/-1
import bootstarp from 'counselor-sdk';

const request = () => {
  // 此处假设getWxconfig是你获取微信配置参数的接口函数
  const {code, data, message} = await getWxconfig()
  // 其中data应该为一个对象包含微信初始化必须的字段，详情看微信公众号开发者文档
  return {code, data, message}

}

const {code} = await bootstarp(request, {
  initCallback: () => {
    // 这里是你需要立马调用的Api位置,无需则不用传该参数
  }
})
