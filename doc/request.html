# 请求类接口（request）

# sendPost

  • 说明：调用平台能力发送 post 请求
  • 参数 payload 对象如下：
    • url {string} 请求地址
    • body {object} 请求参数
    • headers {object} 请求头
  • tips：此 Api 微信无实现，其中 url 参数需要绝对路径
  • 返回值：
    • 成功：{sdkApiReturnSuccessResponse}
    • 失败：{sdkApiReturnErrorResponse|sdkEnvErrorResponse}
  • code 返回范围：0/-1/-7
import { sendPost } from "counselor-sdk";

const { code, data } = await sendPost({
  url: "http://xxx.xxx.com/getXXX",
  body: {
    a: 1,
    b: 2,
  },
  headers: {
    xxx: "xxx",
  },
});

if (code === -7) {
  // 请自行发送post请求
}

if (code === 0) {
  // 成功时返回 {code, data, message}对象
  const { code: postCode, data: postData, message: postMessage } = data;
}
